# Door-RFID
用 python3 RFID 一下

## Instructions

- install packages
```
pip3 install -r requirements.txt
```

- set virtual environment
```
virtualenv doorrfid
```

- open virtual environment
```
source doorrifd/bin/activate
```

- pip3 install packages
```
pip3 install xxxxx
```

- pip3 save packages
```
pip3 freeze > requirements.txt
```

- run program
```
python3 index.py
```

- close virtual environment
```
deactivate
```
